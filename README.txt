WiFi Password Viewer
======================

This desktop application allows you to view all saved Wi-Fi passwords on your PC in an interactive and user-friendly GUI.

----------------------
Created by: QuickSilver
Date: September 4th, 2024
----------------------

Repository Name: WiFi

Description:
-----------------
The Wi-Fi Password Viewer is a Python-based GUI application built using tkinter. It fetches all the saved Wi-Fi profiles and displays their passwords. This can be particularly useful when you need to retrieve saved Wi-Fi passwords on your PC.

Features:
-----------------
- Simple and interactive GUI using tkinter.
- Fetches all saved Wi-Fi network names and passwords.
- Displays network information in a readable format.

How to Use:
-----------------
1. Install Python from https://www.python.org if you don't have it already.
2. Run the `wifi_password_viewer.py` script using the following command:

                      python wifi_password_viewer.py

3. Click on the "Show Saved Wi-Fi Passwords" button to view your saved Wi-Fi profiles and their passwords.

Requirements:
-----------------
- Python 3.x
- tkinter (pre-installed with Python)
- subprocess module (used for running system commands)
